PayDock

PayDock is an open-source SDK for iOS and a solution for collecting and handling payment sources in secure way through PayDock.

PayDock is a smart payments platform connecting to your e-commerce or online payments solution/gateway. We enable you to add multiple payment gateways instantly, access new payment methods and manage all payments data from one place.
